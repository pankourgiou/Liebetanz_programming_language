Liebetanz Syntax

Each line is one spell. A spell is a word followed by its arguments.

// single-arg spells verdant 9 → 3 fakultät 5 → 120 mondkurve 1.5708 → 1 // two-arg spells zwirn 2 8 → 256 asche 17 5 → 2 ggt 48 18 → 6 // bind a name bind x = zwirn 3 4 → x = 81 // use a bound name verdant x → 9 // chain with | silber | zwirn 10 3 → log₁₀(1000) = 3

Blank lines are ignored. # starts a comment.

Click any spell in the spellbook panel to insert it into the editor.
